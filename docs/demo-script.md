# Demo script (3–5 минут)

> Если видео не сдаёшь, этот файл можно использовать как чеклист ручной демонстрации фич.

## 0:00–0:30 — Вступление
- Открыть лендинг.
- Коротко: «WishWave убирает дубли подарков и сохраняет сюрприз для владельца списка».

## 0:30–1:40 — Owner flow
- Войти/зарегистрироваться.
- Создать wishlist.
- Добавить 2 товара:
  - `single` (резерв),
  - `group` (совместный сбор).
- Для одного товара вставить URL Ozon/WB и показать автозаполнение.
- Опубликовать список и скопировать публичную ссылку.

## 1:40–3:40 — Guest + realtime
- Открыть ссылку в инкогнито.
- Создать guest-session.
- Зарезервировать single-item.
- Внести вклад в group-item.
- Показать, что у owner всё обновляется без перезагрузки.

## 3:40–4:30 — Proof of privacy
- В owner-режиме показать:
  - есть агрегаты (reserved/progress),
  - нет имён гостей и персональных сумм.

## 4:30–5:00 — Технический финал
- Упомянуть стек: Next.js + FastAPI + PostgreSQL + WebSocket.
- Коротко показать README, тесты и деплой.
